# NoSQL Projekt

Projekt za obradu studentskih podataka.

## Kako pokrenuti prvi zadatak

### Učitavanje Podataka

```bash
node prvi/main.js -load
```

### Zadaci (z1 - z8)

**Z1: Zamjena Nedostajućih Vrijednosti**

```bash
node prvi/main.js -z1
```

**Z2: Statistička Analiza Kontinuiranih Varijabli**

```bash
node prvi/main.js -z2
```

**Z3: Frekvencijska Analiza Kategoričkih Varijabli**

```bash
node prvi/main.js -z3
```

**Z4: Dijeljenje po Srednjoj Vrijednosti**

```bash
node prvi/main.js -z4
```

**Z5: Embediranje Frekvencija**

```bash
node prvi/main.js -z5
```

**Z6: Embediranje Statistike**

```bash
node prvi/main.js -z6
```

**Z7: Identifikacija Prekoračenja Praga Standardne Devijacije**

```bash
node prvi/main.js -z7
```

**Z8: Složeni Indeks i Query Optimizacija**

```bash
node prvi/main.js -z8
```
